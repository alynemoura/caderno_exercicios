📋 Sobre o Capítulo
Este repositório contém a implementação prática dos exercícios do Capítulo 08, que aborda conceitos fundamentais de manipulação de arrays com NumPy. O material foi desenvolvido para consolidar o aprendizado sobre operações vetoriais, indexação avançada, broadcasting e manipulação de matrizes.

🎯 Tópicos Abordados
Operações Vetoriais: Criação e manipulação de arrays unidimensionais

Indexação e Slicing: Técnicas avançadas para seleção de elementos

Broadcasting: Como o NumPy lida com arrays de diferentes formas

Manipulação de Matrizes: Operações com arrays multidimensionais

Funções Universais (ufuncs): Operações elemento a elemento
